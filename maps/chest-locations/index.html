Common Chest

2 Primogems each
Found everywhere
Total: ~500+ per region

Exquisite Chest

2-5 Primogems each
Often behind puzzles
Total: ~200+ per region

Precious Chest

5 Primogems each
Reward for challenges
Total: ~50+ per region

Luxurious Chest

5 Primogems each
Rare, major rewards
Total: ~20+ per region
