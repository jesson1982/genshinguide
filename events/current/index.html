Current Events

All active events and how to complete them efficiently

🎉 How to Check Active Events

In-Game Events Menu

Press F4 (PC) or check the Events menu to see all active events with timers and rewards.

Event Types

Combat Challenges, Exploration Events, Mini-Games, Login Bonuses, and Story Quests.

💰 Typical Event Rewards

Reward TypeAmount
Primogems (Major Event)420-1,000+
Primogems (Minor Event)100-300
Crown of Insight1 per major event
Weapon/Artifact MaterialsVaries
Mora & Enhancement Ores50,000+ Mora

🎮 Event Completion Tips

Prioritize Events

Complete events with Primogem rewards first. Some events are time-limited.

Use Trial Characters

Many combat events provide trial characters. Use them if you lack strong units.

Check Daily Reset

Some events have daily point caps. Log in daily to maximize rewards.
