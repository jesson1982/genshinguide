Element Reactions Guide

Master all elemental reactions for maximum damage

🔥 Amplifying Reactions (Damage Multipliers)

🔥+💧

Vaporize

Pyro + Hydro

Pyro on Hydro = 1.5x | Hydro on Pyro = 2x

Best Teams: Hu Tao + Xingqiu/Yelan

🔥+❄

Melt

Pyro + Cryo

Pyro on Cryo = 1.5x | Cryo on Pyro = 2x

Best Teams: Ayaka + Bennett

⚡ Transformative Reactions (Fixed Damage)

🔥+⚡

Overloaded

Pyro + Electro

AoE Pyro damage + knockback

Best For: Crowd control, shield breaking

💧+⚡

Electro-Charged

Hydro + Electro

Continuous Electro DoT

Best Teams: Raiden + Xingqiu

🌀+Any

Swirl

Anemo + Any Element

Spreads element + Viridescent RES shred

Best For: Kazuha/Venti support builds

🌿 Dendro Reactions (Version 3.0+)

💧+🌿

Bloom

Hydro + Dendro

Creates Dendro Cores that explode

Best Teams: Hyperbloom with Raiden

⚡+🌿

Aggravate

Electro + Dendro

Increases Electro damage

Best Teams: Keqing + Nahida

🌿+⚡

Spread

Dendro + Electro

Increases Dendro damage

Best Teams: Alhaitham + Fischl

💧 Utility Reactions

💧+❄

Freeze

Hydro + Cryo

Immobilizes enemies

Best Teams: Ayaka + Mona freeze comp

💎+Any

Crystallize

Geo + Any Element

Creates elemental shields

Best For: Zhongli/Navia teams
